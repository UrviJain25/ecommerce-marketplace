Ecommerce-marketplace

A full-stack multi-vendor e-commerce platform where users can browse products, add to cart, and place orders, while sellers manage inventory and admins monitor sales.

Features
User
- Register & Login (JWT Authentication)
- Browse Products
- Search & Filter
- Add to Cart
- Checkout & Orders

Seller
- Add / Update / Delete Products
- Manage Inventory

Admin
- Sales Analytics
- Top-Selling Products
- Reports


Advanced Concepts
- Transactions (Checkout Flow)
- SQL Joins & Aggregations
- Triggers (Stock Update)
- Views (Sales Reports)
- Window Functions (Top Products)
- Role-Based Access Control (RBAC)

Tech Stack
Frontend:React + TypeScript + Redux + Tailwind  
Backend:FastAPI (Python)  
Database:MySQL / PostgreSQL  
Deployment:Azure  

Sample APIs
* POST `/register`
* POST `/login`
* GET `/products`
* POST `/cart`
* POST `/checkout`
